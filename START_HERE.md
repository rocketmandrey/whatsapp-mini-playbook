# 🚀 START HERE — как дать своему ИИ руки в WhatsApp

Тебе **не нужно ничего знать про код**. Нужно дать своему AI-агенту (Claude Code или
Cursor) правильную команду — дальше он ведёт по шагам. Ты только подтверждаешь и
сканируешь QR телефоном.

> Что вообще делаем: поднимаем на твоём сервере **мост** к твоему WhatsApp (он копит
> архив всех чатов) и **MCP-сервер** поверх него. После этого любой твой Claude (на
> ноуте, на телефоне) умеет читать твои переписки, искать по ним и слать сообщения —
> обычными словами, без кода.

---

## Шаг 0 — скачай репозиторий себе

На странице репо в GitHub: зелёная кнопка **Code → Download ZIP** → распакуй.
(Если знаешь git: `git clone <url этого репо>`.)

**Никогда не открывала терминал (Mac):** нажми ⌘+пробел, набери «Терминал», Enter.
В окне набери `cd ` (с пробелом), перетащи мышкой распакованную папку прямо в окно
терминала, нажми Enter — ты «внутри папки».

**Claude Code ещё не установлен:** команда установки — на официальной странице
[claude.com/claude-code](https://claude.com/claude-code) (подписка Pro подходит).
Поставил → набери в терминале `claude`, Enter — откроется диалог, туда и вставишь
промпт из Шага 1.

Сервер и домен покупать заранее НЕ нужно — агент проведёт по покупке сам
([`02_WHATSAPP_MCP.md §0`](02_WHATSAPP_MCP.md)).

## Шаг 1 — скопируй промпт агенту

Открой Claude Code / Cursor в папке этого репозитория и вставь:

```
Ты — мой оркестратор по подключению личного WhatsApp к нейронке через MCP.
Прочитай этот репозиторий, особенно: AGENTS.md, 01_SECURITY.md,
02_WHATSAPP_MCP.md и деплой-комплект examples/wa-mcp/.

Дальше веди меня ШАГ ЗА ШАГОМ (по одному шагу, спрашивай перед каждым):
1) выясни вводные: какие клиенты подключаем (Claude Code на ноуте / телефон /
   оба); сервер у меня есть или поднимаем; какой номер вешаем (предупреди,
   что лучше отдельный, см. 01_SECURITY.md §1);
2) ПРОВЕРЬ ПРЕРЕКВИЗИТЫ: VPS (root, тариф 2 ГБ RAM — мост тянет Chromium),
   поддомен wa-mcp.* и A-запись на IP (если нужен доступ с телефона);
   если сервера или домена НЕТ — сначала пройди со мной 02_WHATSAPP_MCP.md §0:
   предложи дефолтный вариант ИМЕННО по ссылкам из §0 (Beget: VPS
   beget.com/p164176/ru/vps тариф 2 ядра/2 ГБ, домен beget.com/p164176/ru/domains);
   ⚠️ ПРОСЛЕДИ, чтобы локация сервера была Европа/Латвия, НЕ Россия — WhatsApp
   с российских IP не работает; дождись покупки, настрой SSH-ключ
   (ssh-keygen → ssh-copy-id) и A-запись, проверь dig;
3) секреты сгенери САМ (openssl rand -hex 24 → BRIDGE_TOKEN и WA_MCP_SECRET),
   впиши в .env на сервере, chmod 600; внешних ключей тут нет;
   НЕ печатай .env и секреты в чат;
4) проверь безопасность по 01_SECURITY.md: .env в .gitignore, порты MCP
   только на 127.0.0.1, план закрыть QR-порт после линковки;
5) подними мост по 02_WHATSAPP_MCP.md из examples/wa-mcp/, дай мне QR,
   дождись подтверждения линковки (curl 127.0.0.1:8799/me → мой номер),
   потом ЗАКРОЙ порт 8795 фаерволом;
6) подними HTTP MCP + nginx + certbot, подключи клиентов: Claude Code
   (claude mcp add --scope user) и claude.ai-коннектор для всех остальных
   устройств (телефон/веб/десктоп) — дай мне готовую ссылку
   https://claude.ai/new?modal=add-custom-connector#settings/customize-connectors;
   URL коннектора содержит секрет — в чат его НЕ печатай: запиши в файл
   connector-url.txt у меня на рабочем столе, скажи скопировать оттуда в поле
   URL формы (на телефон — переслать себе файлом/заметкой), после добавления
   предложи удалить файл; прогони смок-тест из 02_ §5 — без него «не готово»;
7) когда заработало — вынеси оба процесса в автозапуск (systemd, шаблоны
   в examples/wa-mcp/deploy/).
В конце дай сводку: что поднято, какие секреты я должен хранить и как их
ротировать при утечке.

Не делай несколько шагов молча. Я не программист — объясняй простыми словами.
Веди меня как поддерживающий напарник: перед шагом — что делаем и зачем,
после шага — что получилось и сколько осталось. Если что-то упало — спокойно:
«это норма, чиним вот так». Сообщения от моего имени не отправляй без явного
подтверждения.
```

## ✅ Что агент сделает за тебя

- проверит prereqs (сервер, RAM, домен, A-запись) и предупредит, чего не хватает;
- сгенерит секреты, развернёт мост + HTTP MCP из готового комплекта `examples/wa-mcp/`;
- настроит nginx + SSL (certbot), спрячет порты на `127.0.0.1`, закроет QR после линковки;
- подключит Claude Code (`--scope user` — доступно из любой папки) и телефон;
- прогонит смок-тест: `whoami` → твой номер, поиск находит, чужой секрет → 404;
- вынесет в автозапуск (`systemd`) с `Restart=always`.

## 🙅 Чего агент НЕ сделает (тут нужен ты)

- не отсканирует за тебя QR — это делает твой телефон (WhatsApp → Связанные устройства);
- не повесит коннектор на твой личный/рабочий номер без предупреждения;
- не закоммитит секреты и `wa_session/` — это запрещено правилами `AGENTS.md`;
- не отправит сообщение от твоего имени без явного подтверждения (по умолчанию — черновик);
- не включит рассылки/массовые действия (прямой путь к бану).

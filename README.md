# 📱 WhatsApp Mini Playbook — твой вотсап как руки твоего ИИ

МИНИ-плейбук: как развернуть **свой** MCP-коннектор к личному **WhatsApp** на своём
сервере — чтобы любой твой AI-клиент (Claude Code на ноуте, Claude-приложение на
телефоне, агент в Cursor) умел читать твои чаты, искать по ним и слать сообщения.
Это выжимка «только WhatsApp» из большого плейбука
[`mcp-messengers-playbook`](https://github.com/rocketmandrey/mcp-messengers-playbook)
(Telegram + WhatsApp) — здесь один путь, короче и без развилок.

> **👉 Не знаешь, с чего начать — открой [`START_HERE.md`](START_HERE.md).** Там
> копипаст-промпт агенту: вставил — и он ведёт тебя по шагам. Кода знать не нужно.

## Обещание

Ты пишешь обычными словами — «догони меня по рабочему вотсап-чату за 3 дня»,
«найди, что клиент писал про сроки», «набросай ответ — отправлю сам» — и агент сам
лезет в твой WhatsApp и действует. Один мост + один MCP-сервер на твоём VPS — и руки
появляются сразу у всех клиентов: ноут, второй комп, телефон.

## Кейсы — не теория, всё из жизни автора

1. **✍️ Переговоры одним промптом.** Реальный случай: дал Клоду коннектор и задачу —
   договориться об аренде оборудования в кофейне для мастер-класса. Дальше цикл такой:
   присылают ответ → открываешь тот же чат → «прочитай и ответь». Нейронка сама сверяет
   ответ с целью переговоров и пишет следующее сообщение. Ты жмакаешь «отправить».
   Фокуса тратится ноль. Тот же приём тащит длинные мутные переписки с
   гос-учреждениями — вежливо, последовательно, до результата.
2. **🧵 Родительский чат на 30 человек. В вацапе. На испанском.** Мост складывает
   сообщения в базу, дважды в день нейронка присылает короткое саммари «что там
   важного». А боту можно задавать вопросы: «что решили про сбор денег?» — он копается
   в базе и отвечает.
3. **🔍 Поиск по перепискам.** «Что мы решили с клиентом в марте?» → агент находит
   и цитирует. Работает по всем чатам сразу — группы и личка.
4. **🧠 Оцифровка продавцов.** Переписки WhatsApp — готовое сырьё: нейронка
   раскладывает их на сценарии («сколько стоит», возражения, воронка) и собирает базу
   знаний компании. Один раз оцифровал — подключаешь эти мозги к любому каналу.
5. **📱 Всё это — с любого устройства.** Коннектор вешается один раз на аккаунт
   claude.ai и работает на телефоне, в вебе и в десктоп-приложении, без ноута.

## Мини-словарь (для тех, кто не кодит)

- **MCP** (Model Context Protocol) — стандарт, которым AI-клиент получает доп-инструменты.
  MCP-сервер = «коробка с инструментами»: подключил — и агент умеет читать/искать/слать.
- **мост (bridge)** — процесс, который держит связь с WhatsApp как «связанное устройство»
  (Linked Device), копит архив чатов в SQLite и умеет отправлять.
- **сессия** (`wa_session/`) — файл-пропуск в аккаунт без логина = пароль. Создаёшь сам
  (сканируешь QR), не шаришь, в репо её нет.
- **секрет-в-URL** — секретная ссылка, в которой сам секрет = ключ доступа. Утёк —
  равно утекла сессия; ротируется одной строкой в `.env`.
- **nginx** — «швейцар»: единственная дверь с улицы к серверу, с https-замочком.
- **certbot** — бесплатный https-сертификат, сам себя продлевает.
- **systemd** — автозапуск: упало или сервер перезагрузился — само поднялось.
- **смок-тест** — короткая проверка «дым не идёт» перед тем, как сказать «готово».

## Что нужно заранее (prereqs)

- **VPS** — Ubuntu, root, тариф **2 ГБ RAM** (мост тянет Chromium), публичный IPv4,
  локация **Европа (Латвия), НЕ Россия** — WhatsApp с российских IP не работает;
  ~500–1000 ₽/мес.
  **Нет сервера — не стоп:** агент купит его с тобой за 10 минут по
  [`02_WHATSAPP_MCP.md §0`](02_WHATSAPP_MCP.md) (хостеры с оплатой картой РФ, какой
  тариф выбрать, SSH-ключ). Путь расписан для VPS; Mac mini 24/7 возможен, но шаги
  автозапуска придётся адаптировать самому.
- **Домен** — поддомен `wa-mcp.твойдомен` + A-запись на IP сервера (нужен для доступа
  с телефона). Домена нет — тоже в §0: купить ~200–400 ₽/год, либо тестово
  `<IP>.sslip.io` вообще без покупки.
- **Телефон с WhatsApp** — отсканировать QR при линковке. Внешних ключей НЕ нужно:
  оба секрета (`BRIDGE_TOKEN`, `WA_MCP_SECRET`) генерятся `openssl rand`.

Итого бюджет: VPS ~500–1000 ₽/мес + домен ~300 ₽/год. Больше платить некому —
API-ключей и подписок этот путь не требует.

## ⚠️ Прочитай ПЕРВЫМ (30 секунд, иначе потеряешь аккаунт или сольёшь переписку)

1. **MCP = полный доступ к аккаунту.** Сервер с твоей сессией читает и шлёт всё, что
   можешь ты. Поднимай на СВОЁМ сервере. По возможности — **отдельный номер**.
2. **Секреты — только в `.env`** (он в `.gitignore`). В репо — только `.env.example`.
3. **`wa_session/` и секрет-в-URL = пароль.** Не шарить, не коммитить. Утёк → ротируй.
4. **Порты MCP — только `127.0.0.1`.** Наружу один nginx с SSL. QR-страницу закрой
   фаерволом сразу после линковки.

Полные гардрейлы — [`01_SECURITY.md`](01_SECURITY.md), читать обязательно.

## ✅ Что этот плейбук покрывает

- Свой WhatsApp-мост (`whatsapp-web.js`) на VPS: архив ВСЕХ чатов в SQLite + отправка.
- HTTP MCP поверх архива: один вход для всех клиентов, секрет-в-URL.
- Подключение Claude Code (`--scope user`) и claude.ai-коннектора — один раз на
  аккаунт, работает на телефоне, в вебе и в десктоп-приложении.
- Безопасность коннектора: сессия, секреты, ротация, порты, анти-бан.
